Simple Drawing App

Android Studio Project

Simple drawing application made for android with android studios using kotlin. Has three different brush sizes and a color picker for different colors. User can set a background image and draw on it. Users can then save their drawings and even send them using whatsapp or email.

Worked with a couple of different libraries while developing this. Dealt with permissions and accessing data on users phone.

Application can be found on play store.
