Holey Moley

School Project

3D Multiplayer brawler. Made as a school project in a team of four people. Two programmers and two artists. I worked on the game as a programmer, doing things like UI, combat and game structure.

The game was inspired by Gangbeasts and the object was to throw other players off the map. There would be weapons spawns on the map that had some different functionality. Hitting players with them enough times would stun them for a few seconds.

The game was finnished but never officially released as we didn't have the time to polish it to a satisfying degree. The game can be found here.
